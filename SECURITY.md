# Security Policy

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report security issues privately via [GitHub's private vulnerability reporting](https://github.com/cloudprobe/debrief/security/advisories/new).

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

I'll acknowledge within 48 hours and aim to ship a fix within 7 days for confirmed issues.
